# C# Health Club Membership Calculator

A Windows Forms desktop app that calculates health club membership costs.

## Features

- Selects a membership type
- Adds optional services such as yoga, karate, and personal training
- Validates the number of months
- Calculates monthly fee and total fee
- Includes clear and exit actions

## Skills Demonstrated

- C#
- Windows Forms
- Input validation with `int.TryParse`
- Event-driven programming
- Basic UI state management

## Run

Open `HealthClub.slnx` in Visual Studio and start the project.
